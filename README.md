# LangChain Structured Output

This repository contains examples of generating **structured outputs** from Large Language Models using LangChain. Structured outputs make LLM responses consistent, reliable, and easy to integrate into applications.

## Topics Covered

* JSON Schema
* Pydantic Models
* TypedDict
* `with_structured_output()`

## Prerequisites

* Python 3.10+
* LangChain
* Pydantic

Install the required packages:

```bash
pip install langchain pydantic
```

## Running the Examples

```bash
python <filename>.py
```

## Purpose

This repository is intended for learning how to generate structured responses from LLMs using different schema definitions and LangChain's structured output capabilities.
